# 通用大模型平台 Go SDK

本项目提供一个 Go SDK，用于与各种大模型平台进行交互。

## 特性

*   通过统一的 Go 接口抽象各平台的特定 API。
*   支持多个大语言模型（LLM）平台。
*   处理多种模型类型。
*   管理各平台的身份验证。
*   提供清晰的请求和响应结构。
*   包含错误处理和日志记录功能。

## 支持的平台

*   火山方舟 (初期支持)
*   阿里百炼 (初期支持)

## 支持的模型类型

*   文本生成
*   多模态
*   推理模型
*   音频理解
*   视频理解
*   视频生成
*   图片处理
*   图片理解
*   图片生成
*   向量模型
*   语音合成
*   语音识别
*   排序模型
